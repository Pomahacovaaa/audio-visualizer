# audio visualizer

A standart audio visualizer built with p5.js.

## Features:
* 9 different audio tracks (press keys 1–9 to switch songs)
* Click to play/pause audio
* Real-time waveform visualizer

## Built with:
* HTML
* CSS
* JavaScript
* p5.js (p5.sound)
